RANDOLUNCH

http://www.randolunch.com

Randolunch is a site that helps you decide where to eat by picking a random restaurant for you based on location and cuisine. Launched in September 2013, I built this site primarily to improve my skills in Ruby on Rails. I also ended up learning a ton about responsive design, Google Analytics, and integrating with various APIs.

Technologies:

  • Ruby on Rails 3.2
  • Twitter Bootstrap 3
  • jQuery
  • Yelp API
  • Google Maps API
